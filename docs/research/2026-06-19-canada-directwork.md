# Canada — direct-work-route factor scoring — 2026-06-19

**Profile:** Bangladeshi (Dhaka) couple, both ~2022 BSc-CSE / software-AI engineers; goal migrate → PR → citizenship → passport. This brief scores the new `direct-work-route` category (the alternative to the study-first route).

## Method

Gov-first targeted research (subagent), BD lens, ≥2 sources per decisive claim, 2025-26 currency. Sources: IRCC Express Entry rounds, ESDC Global Talent Stream, 2026–28 Levels Plan, Bangladesh Embassy DNC; cross-checked with CIC News on the STEM-category change. canada.ca pages 403 the fetch tool — figures verified via search extracts + reputable secondary sources.

## Decisive findings

- **No blocker.** Dual citizenship allowed both ways; Canada is on Bangladesh's DNC list → naturalising Bangladeshi retains BD nationality.
- **The headline tech fast-track is gone for this profile:** software engineer (NOC 21231) and data scientist (NOC 21211) were removed from the STEM category-based draws in Feb 2025, and no STEM draw has run since Apr 2024. Arranged-employment CRS points were abolished (Mar 2025).
- **Work permit is fast, PR is the bottleneck:** Global Talent Stream issues software/data work permits in ~2 weeks (employer-driven), but PR conversion needs a high general/CEC Express Entry CRS (~485–547 in 2025).
